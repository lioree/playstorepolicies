# Privacy Policy

**Count Up** (com.elkosoft.countup)

Last updated: July 10, 2026

This Privacy Policy describes how the Count Up app, developed by Lior Elkotser ("the developer"), handles user data.

## Data Collection

Count Up is a local timer application. The app does not collect, store, or transmit any personal data. It does not require account creation and does not transmit timer sessions to a remote server.

## Data Processed on Your Device

The timer value and its running state are processed on your device so the application can display and resume your current session while the app is open. This data never leaves your device.

## Advertising, Analytics, and Tracking

Count Up does not include advertising, analytics SDKs, or background tracking.

## Third-Party Services

Android system components may still process technical information such as crash logs, install data, or device diagnostics according to your device settings and Google Play services configuration. This processing is governed by Google's privacy policy, not by this app.

## Data Sharing

Count Up does not share any user data with third parties, because it does not collect any.

## Changes to This Policy

If a future version adds cloud sync, analytics, or account features, this policy will be updated before those features are released.

## Contact

If you have questions about this Privacy Policy, contact the developer at lioree@gmail.com.
